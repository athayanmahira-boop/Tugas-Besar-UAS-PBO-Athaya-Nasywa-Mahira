# Tugas Besar PBO – Sistem Buku Daftar Akta PPAT

## Deskripsi
Aplikasi berbasis Java Console untuk pencatatan data Akta PPAT dan Pajak Akta  
menggunakan konsep Pemrograman Berorientasi Objek (PBO) dan database MySQL (JDBC).

## Fitur Utama
- Login pengguna (Admin & Staff)
- CRUD Data Akta
- CRUD Data Pajak Akta (SSP & SSB)
- Rekapitulasi tahunan (COUNT, SUM, AVG)
- Validasi input & exception handling

## Konsep PBO yang Digunakan
- Class & Object
- Constructor
- Inheritance (Pajak → SSP & SSB)
- Interface & Implementasi (DAO)
- Collection (List)
- Exception Handling

## Teknologi
- Java
- MySQL
- JDBC
- phpMyAdmin

## Struktur Folder
- src/ → source code program
- lib/ → library JDBC MySQL

## Cara Menjalankan
1. Import database ke MySQL
2. Atur koneksi di DBConnection.java
3. Jalankan MainApp.java
4. Program dijalankan melalui terminal/console
